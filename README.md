# core-GUI

##Requirements
1. NodeJS

## Instructions:
1. `git clone https://github.com/happyastronaut/core-GUI.git`
2. `cd core-GUI`
3. `npm i npm electron --save-dev`
4. `npm i @cityofzion/neon-js`
5. `npm i --save electron-config`
6. `npm i systeminformation --save`
7. `npm i electron-drag`

##RUN
1. `npm start`
